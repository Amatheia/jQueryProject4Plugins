How the plugin works:

The jQuery Validation Plugin makes simple clientside form validation easy. The plugin comes bundled with a useful set of validation methods, while providing an API (Application Program Interface specifies how software components should interact) to write your own methods.

  • Library of validation methods (such as emails, URLs (the address of a World Wide Web page), credit card numbers).
  • Place error messages in the DOM (The Document Object Model - this means creating a model in which the document or web page contains objects (elements, links, etc.) that can be manipulated) and show and hide them when appropriate.
  • Plugin methods:
    • validate() – Validates the selected form.
    valid() – Checks whether the selected form or selected elements are valid.
    rules() – Read, add and remove rules for an element.
  • Custom selectors (selectors are based on CSS3).
  • List of built-in Validation methods: A set of standard validation methods is provided.

Include these lines in your html page:
script src="https://ajax.googleapis.com/ajax/libs/jquery/3.2.1/jquery.min.js"
script src="https://cdnjs.cloudflare.com/ajax/libs/jquery-validate/1.16.0/jquery.validate.js"
Your own js file: script src="jqueryvalidation.js" type="text/javascript" charset="utf-8"
And other scripts and css that suit your goals.
